# 04-施設向け説明トーク

> ポジション：**「保証人代行」ではなく「現場負担軽減パートナー」**  
> 対象：施設長、生活相談員、ケアマネ（電話・訪問・オンライン共通）

---

## テレアポ冒頭（30秒）

```
お忙しいところ失礼いたします。○○（事業名）の△△と申します。
今日は営業というより、施設様が「身寄りのない方・緊急連絡先が実質いない方」の入所や運用で
実際に困っていることをお聞きしたく、ご連絡しました。
15〜20分ほど、現場のお困りごとについて教えていただけないでしょうか。
```

**避ける表現：** 「保証人がいない方を必ず入所させます」「医療同意も代わりにできます」

---

## 本題への導入（1分）

```
厚生労働省も、身元保証人がいないことだけを理由に入院や入所を断ることは
適切ではないと整理されています。
ただ現場では、費用の支払い、急変時の連絡、退所後の行き先、死亡時の対応など、
家族にやってもらっていた部分がなくて困る、というお声をよく伺います。

私たちは、その「家族に近い役割」のうち、契約で明確にできる部分を
外部のパートナーとして担うサービスを検討しています。
```

---

## サービス概要（2分）

```
具体的には、例えば次のような範囲です。

・緊急連絡先としての登録と、急変時・入退所時の連絡
・月次費用などの支払い代行（事前にお預かりした預託金からの精算）
・入退院・入退所の手続き同行
・診療説明などの同席（※医療同意そのものは本人の意思が基本です）
・事前契約に基づく死後事務の連絡・手配の仲介

一方で、医療同意の代行や、無制限の連帯保証など、
法的・倫理的に危険な部分は明確に「対応しない」としています。
施設様が安心して紹介できるよう、契約内容と重要事項説明書で範囲を開示します。
```

---

## 施設メリット（1分）

```
施設様にとってのメリットは、大きく3つです。

1. 入所判断の際、「誰に連絡すればよいか」「費用は誰が払うか」が契約で明確になる
2. 急変時・夜間の連絡先が一本化され、現場の負担が減る
3. 死亡時・残置物など、施設だけで費用や手間を負担しがちな場面を事前契約で整理できる

営業で無理に入所者を増やすのではなく、
すでに困っているケースの「仕組み」を整えるイメージです。
```

## よくある質問への回答

### Q. 身元保証人と同じですか？

```
完全に同じではありません。法的な連帯保証の範囲は契約で限定し、
主に「連絡・支払い代行・手続き支援・死後事務の事前契約」に焦点を当てています。
施設様が困りやすい実務の部分を、契約で見える化するサービスです。
```

### Q. 医療同意もやってもらえますか？

```
いいえ、医療同意は本人の身体・生命に関わる一身専属的な権利です。
私たちは説明の同席や、本人が理解しやすいよう補足することはできますが、
同意の代行は行いません。ここは明確に線を引いています。
```

### Q. 費用は誰が払いますか？

```
基本はご本人（または成年後見人等）の月額料金と、必要に応じた預託金です。
施設様への紹介料などは、現在検討中で、施設様のご負担にならない形を目指しています。
```

### Q. 預託金はいくらですか？返金は？

```
金額・返金条件は重要事項説明書で事前にお渡しします。
運転資金と分けて管理し、使途と残高を定期報告する設計にしています。
```
