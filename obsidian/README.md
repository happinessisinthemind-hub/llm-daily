# Obsidian への取り込み方法

## まとめ一式（Notion・Obsidian 共通）— おすすめ

**1ファイルで OK：**

```
export/高齢者支援事業-まとめ一式.md
```

| 方法 | 手順 |
|------|------|
| **貼り付け** | ファイルを開く → 全文コピー → Obsidian 新規ノート → 貼り付け |
| **コピー** | `.md` を iCloud Vault に保存 |

目次 `[[#パート1...]]` から各セクションへジャンプできます。

---

## 個別ノート版（ウィキリンク付き）

`obsidian/高齢者支援事業/` フォルダを Vault にコピー。`00-索引（MOC）` から辿ります。

## iCloud Vault へ入れる

1. Obsidian → **VaultをFinderで開く**
2. `export/高齢者支援事業-まとめ一式.md` または `obsidian/高齢者支援事業/` をコピー

詳細: `export/README.md` / `docs/integration/obsidian-notion-sync.md`
