# 高齢者終身サポート事業 — 面談・初月成果物

## 推奨ポジション

- **A：サービス設計パートナー（70%）**
- **C：現場アドバイザー（30%）**
- **B：営業支援** — ヒアリング設計・トーク台本監修・必要時同席まで
