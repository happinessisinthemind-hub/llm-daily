# パート1：事業設計整理書（15項目）

# 入退院・施設入居あんしん調整サポート — 事業設計整理

> 身寄りのない高齢者向け「入退院・施設入居支援サービス」市場調査まとめ（Hermes / OpenClore 調査＋現場知見）を、事業設計・面談・提案書に転用しやすい形に再構成したもの。

**結論（一行）**：市場性はある。ただし初期は「身元保証を丸ごと担う事業」ではなく、「保証人不在で止まっている調整を進める相談調整サービス」から始めるのが現実的。

---

## 1. 事業概要

### 事業の位置づけ

介護・高齢者福祉領域において、身寄りのない高齢者（保証人・緊急連絡先・死後対応者がいない高齢者）が、入院・施設入居・退院・死亡時などの場面で行き場を失ったり、支援が停滞したりする課題を解決する新規サービス。

### 募集主の背景

- 損害保険会社出身。リスク・保証の設計経験あり
- 介護現場のリアル、病院・施設・ケアマネ・相談員側の実務感覚には現場経験者の知見が必要

### 推奨する事業名・表現

| 区分 | 内容 |
|------|------|
| 推奨サービス名 | **入退院・施設入居あんしん調整サポート** |
| 避ける名称 | 「身元保証サービス」（責任範囲が広く見えすぎる） |
| 売り方（OK） | 保証人不在で止まっている調整を進める |
| 売り方（NG） | 保証します／家族代わりになります |

### 初期の事業モデル

```
相談調整サービス（自社）
  ＋ 士業・社協・終身サポート事業者との連携（外部）
```

最初は保証しない・預託金を預からない・医療同意しない・死後事務を一人で抱えない。

---

## 2. 解決する社会課題

### 顕在化する困りごと

身寄りのない高齢者が、以下の理由で入院・入所・退院・死後対応が滞る。

- 保証人がいない
- 緊急連絡先がいない
- 退院時の引き取り手がいない
- 死後対応を担う人がいない
- 金銭管理や契約手続きの支援者がいない

### 制度と現場のズレ（事業ニーズの本質）

| 制度上 | 現場実務上 |
|--------|------------|
| 身元保証人不在のみを理由とした入院・入所拒否は不適切 | 緊急連絡・支払い・退院後行き先・死亡時対応・残置物など「家族機能」がなく困る |
| 医師法・介護施設の法令整理 | 誰に連絡するか・誰が同意するか・誰が支払うかが曖昧で止まる |

**キーフレーズ**：保証人がいないから入れない、ではなく、**家族機能が分解されていない**ことが現場の詰まり。

### 需要が存在する背景

- 高齢単身世帯の増加
- 家族と疎遠な高齢者の増加
- 入院・入所時の保証人・緊急連絡先要求の常態化
- 退院調整・死亡時対応者不在の増加

## 3. 想定顧客

### 利用者

| セグメント | 特徴 |
|------------|------|
| 独居高齢者・身寄りのない高齢者 | 主ターゲット |
| 家族と疎遠な高齢者 | 遠方家族が支払う可能性あり |
| 資産・年金があるが身寄りがない高齢者 | 有料化しやすい |
| 認知症・判断能力に懸念がある高齢者 | 個別設計・専門職連携必須 |
| 生活保護受給者・低所得層 | 民間有料だけでは困難。行政・社協連携が必要 |

### 支払者（優先順）

1. **本人**（いちばん現実的）
2. 遠方家族・疎遠だが最低限支援したい親族
3. 成年後見人・士業経由の本人資産
4. 居宅介護支援事業所等（法人向け顧問・研修）
5. 施設・病院の直接負担（弱い。紹介・連携はあるが支払者にはなりにくい）
6. 行政・社協（低所得層向けは委託・連携モデル）

### 紹介者（最初に営業すべき順）

1. **ケアマネジャー**
2. **地域包括支援センター**
3. **施設相談員**
4. **MSW（医療ソーシャルワーカー）**

※ 病院MSWはニーズは強いが多忙・警戒感も強い。最初から病院突撃より、上記4者で信頼を作る。

### 提携先

| 種別 | 役割 |
|------|------|
| 弁護士・司法書士・行政書士 | 法的判断・契約・相続 |
| 社会福祉協議会・地域包括 | 低所得層・地域連携 |
| 終身サポート事業者 | 身元保証・死後事務（専門領域） |
| 成年後見に詳しい専門職 | 判断能力・後見制度 |
| 葬儀社・遺品整理業者 | 死後事務の実務 |
| 自治体 | 生活保護・行政手続き |

- ケアマネ・MSW・施設相談員との調整

- 現場が安心して紹介できる説明資料・情報シート

## 5. 提供しない方がよいサービス（初期段階）

以下は初期では**明確に線を引く**。

| 提供しないこと | 理由 |
|----------------|------|
| 身元保証人・連帯保証の丸ごと引受 | 法的責任・倫理リスクが高い |
| 預託金の自社預かり | 返金トラブル・利用者保護 |
| 医療同意の代行 | 第三者に同意権限なし |
| 「家族代わり」のうたい文句 | 責任範囲が際限なく広がる |
| 死後事務の単独抱え込み | 相続人トラブル・専門性 |
| 金銭管理の自社一括 | 利益相反・監査リスク |
| 24時間対応の安請け合い | 体制なしは契約不履行・炎上 |
| 紹介料・キックバックモデル | 利益誘導・中立性の喪失 |
| 判断能力に不安な人との単独契約 | 契約無効・トラブル |
| 法的判断の無資格実施 | 士業領域 |

## 6. 法的・倫理的リスク

### 特に危険な設計

- 医療同意を代行できると誤解させる
- 預託金を自社資金と混ぜる
- 判断能力低下者と不適切に契約
- 死後事務で相続人と揉める
- 施設・病院への紹介料で利益相反
- 保証・金銭管理・生活支援・死後事務を1社で抱えすぎ
- できること・できないことを曖昧にする

### 注意すべき領域（チェックリスト）

- [ ] 医療同意（本人意思が基本）
- [ ] 身元保証・連帯保証の範囲限定
- [ ] 預託金の分別管理
- [ ] 死後事務・相続
- [ ] 成年後見との関係
- [ ] 判断能力確認

- [ ] 紹介料・キックバック
- [ ] 契約時の重要事項説明（消費者庁ガイドライン準拠）

### 紹介者が安心する条件

料金透明、契約範囲明確、医療同意を勝手にしない、預託金管理が安全、士業・社協・地域包括と連携、レスポンスが早い、中立性、説明資料が分かりやすい。

## 7. 競合・料金相場

### 競合の種類

- 民間身元保証・終身サポート事業者
- NPO・社協系サービス
- 士業連携サービス
- 成年後見関連サービス
- 葬儀・死後事務関連

### 既存サービスの料金相場（ざっくり）

| 項目 | 相場 |
|------|------|
| 初期費用 | 80万〜150万円前後 |
| 預託金 | 50万〜100万円前後 |
| 月額 | 5,000円〜1万円前後 |
| 死後事務・葬儀 | 別途 |
| 総額 | 100万〜200万円超も |

**示唆**：高額帯は「資産はあるが身寄りがない」層向け。低所得層は行政・社協連携が必須。枚方市事例（相談88件・契約0件）は、要件・預託金・説明方法の設計が契約率を左右する。

### 差別化の方向

高額な「保証丸ごと」ではなく、**低リスク・スポットの調整支援**から入り、必要に応じて既存保証事業者へつなぐ。

## 11. 面談で話すべきポイント

### 結論文（そのまま使える）

> この事業は市場性があると思います。ただし、最初から身元保証・金銭管理・死後事務まで抱えると、法的・倫理的リスクが高くなります。そのため、初期段階では「身寄りのない高齢者の入退院・施設入居前の情報整理と調整支援」に絞り、士業や既存の保証事業者と連携する形が現実的だと思います。

### 刺さる一文

> 保証人がいないから入れない、ではなく、現場が実際に困るのは、費用精算、緊急連絡、退院・退所後の行き先、死亡時対応、残置物、本人意思確認、医療・介護の説明同席などの"家族機能"が分解されていないことだと思います。

### 言い換え表

| NG | OK |
|----|-----|
| 身元保証をします | 保証人不在で止まっている調整を進めます |
| 家族代わりになります | 本人・家族・病院・施設・ケアマネ・士業の間で情報を整理し、調整を支援します |
| 保証できます | 現場が判断しやすい情報をまとめ、必要な専門職につなぎます |

### 初月で提案できる成果物

1. 現場課題マップ
2. 施設ヒアリングシート
3. サービス範囲表
4. 施設向け説明トーク
5. リスクチェックリスト

### 危険サイン（見極め）

- 「医療同意も家族代わりにできます」
- 預託金管理が曖昧
- 契約書・重要事項説明書が未整備
- 仮払い前の無償作業要求
- CW外の直接契約誘導

## 13. 事業化の勝ち筋

1. **ニッチを絞る**：入退院・施設入居前の情報整理パックから開始
2. **紹介者ファースト**：ケアマネ・地域包括・施設相談員で信頼を積む
3. **中立性**：紹介料なし、できること・できないことを明示
4. **連携モデル**：保証・死後事務は専門事業者・士業へ
5. **現場言語**：「保証」ではなく「調整」「情報整理」「詰まり解消」
6. **低額スポット検証**：3万〜10万円で需要・時間・支払者を実測
7. **資料の質**：紹介者が安心して渡せる説明資料・情報シート
8. **現場経験者の参画**：設計段階からリスクと実務のズレを潰す

## 04-施設向け説明トーク

# 04-施設向け説明トーク

> ポジション：**「保証人代行」ではなく「現場負担軽減パートナー」**  
> 対象：施設長、生活相談員、ケアマネ（電話・訪問・オンライン共通）

## テレアポ冒頭（30秒）

```
お忙しいところ失礼いたします。○○（事業名）の△△と申します。

実際に困っていることをお聞きしたく、ご連絡しました。
15〜20分ほど、現場のお困りごとについて教えていただけないでしょうか。
```

**避ける表現：** 「保証人がいない方を必ず入所させます」「医療同意も代わりにできます」

## 本題への導入（1分）

```
厚生労働省も、身元保証人がいないことだけを理由に入院や入所を断ることは
適切ではないと整理されています。
ただ現場では、費用の支払い、急変時の連絡、退所後の行き先、死亡時の対応など、
家族にやってもらっていた部分がなくて困る、というお声をよく伺います。

私たちは、その「家族に近い役割」のうち、契約で明確にできる部分を
外部のパートナーとして担うサービスを検討しています。
```

---

## サービス概要（2分）

```
具体的には、例えば次のような範囲です。

・緊急連絡先としての登録と、急変時・入退所時の連絡
・月次費用などの支払い代行（事前にお預かりした預託金からの精算）
・入退院・入退所の手続き同行
・診療説明などの同席（※医療同意そのものは本人の意思が基本です）
・事前契約に基づく死後事務の連絡・手配の仲介

一方で、医療同意の代行や、無制限の連帯保証など、
法的・倫理的に危険な部分は明確に「対応しない」としています。
施設様が安心して紹介できるよう、契約内容と重要事項説明書で範囲を開示します。
```

## 施設メリット（1分）

```
施設様にとってのメリットは、大きく3つです。

1. 入所判断の際、「誰に連絡すればよいか」「費用は誰が払うか」が契約で明確になる
2. 急変時・夜間の連絡先が一本化され、現場の負担が減る
3. 死亡時・残置物など、施設だけで費用や手間を負担しがちな場面を事前契約で整理できる

営業で無理に入所者を増やすのではなく、
すでに困っているケースの「仕組み」を整えるイメージです。
```

## よくある質問への回答

### Q. 身元保証人と同じですか？

```
完全に同じではありません。法的な連帯保証の範囲は契約で限定し、
主に「連絡・支払い代行・手続き支援・死後事務の事前契約」に焦点を当てています。
施設様が困りやすい実務の部分を、契約で見える化するサービスです。
```

### Q. 医療同意もやってもらえますか？

```
いいえ、医療同意は本人の身体・生命に関わる一身専属的な権利です。
私たちは説明の同席や、本人が理解しやすいよう補足することはできますが、
同意の代行は行いません。ここは明確に線を引いています。
```

### Q. 費用は誰が払いますか？

```
基本はご本人（または成年後見人等）の月額料金と、必要に応じた預託金です。
施設様への紹介料などは、現在検討中で、施設様のご負担にならない形を目指しています。
```

### Q. 預託金はいくらですか？返金は？

```
金額・返金条件は重要事項説明書で事前にお渡しします。
運転資金と分けて管理し、使途と残高を定期報告する設計にしています。
```
