# Notion への取り込み方法

## いちばん簡単：まとめ一式を1回でインポート（おすすめ）

1. Notion で新規ページを作成（タイトル例：**高齢者支援事業**）
2. ページ右上 `···` → **Import** → **Markdown**
3. 次のファイルを選択：

```
notion/高齢者支援事業-まとめ一式.md
```

これで **事業設計15項目・面談チートシート・初月成果物5点** が1ページに入ります。

---

## ページを分けて入れる場合

`notion/高齢者支援事業/` 内の個別 `.md` をインポート：

```
notion/高齢者支援事業/
├── 00-索引.md
├── 事業設計整理書.md
├── 面談当日チートシート.md
└── 初月成果物/
    ├── 01-現場課題マップ.md
    ├── 02-施設ヒアリングシート.md
    ├── 03-サービス範囲表.md
    ├── 04-施設向け説明トーク.md
    └── 05-リスクチェックリスト.md
```

4. 子ページとして整理し、索引からリンクを貼る

## Cursor Notion MCP で自動作成する場合

1. Cursor デスクトップ → Settings → MCP → **Notion** → Connect（認証）
2. チャットで「`notion/高齢者支援事業` の内容をNotionにページ作成して」と依頼

## Obsidian との関係

- **Obsidian** = iCloud Vault に `obsidian/高齢者支援事業/` をコピー（ウィキリンクあり）
- **Notion** = 本フォルダをインポート（共有・DB向け）
- 詳細: `docs/integration/obsidian-notion-sync.md`
