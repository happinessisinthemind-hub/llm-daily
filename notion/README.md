# Notion への取り込み方法

## まとめ一式（Notion・Obsidian 共通）

**1ファイルで OK：**

```
export/高齢者支援事業-まとめ一式.md
```

1. Notion で新規ページ「**高齢者支援事業**」を作成
2. `···` → **Import** → **Markdown**
3. 上記ファイルを選択

事業設計15項目・面談チートシート・初月成果物5点が1ページに入ります。

---

## ページを分けて入れる場合

`notion/高齢者支援事業/` 内の個別 `.md` をインポート。

## Cursor Notion MCP

クラウドエージェントからは Notion に直接書き込めません。デスクトップ Cursor で認証済みなら、同ファイルのインポートを依頼できます。

詳細: `export/README.md`
