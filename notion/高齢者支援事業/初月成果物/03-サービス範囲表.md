# 03-サービス範囲表

> 目的：「対応する／対応しない／専門職につなぐ」を明文化し、営業・契約・利用者説明のブレを防ぐ。  
> 消費者庁ガイドライン（身元保証等サービス）の重要事項説明と整合させる。

---

## 凡例

| 区分 | 意味 |
|------|------|
| ✅ 対応する | 本サービスの契約範囲内で実施 |
| ⚠️ 条件付き | 事前契約・預託金・本人意思確認等の条件あり |
| ❌ 対応しない | 本サービスでは実施しない（誤解防止のため明記） |
| 🔗 専門職につなぐ | 紹介・同行・情報提供まで。実務は専門家が実施 |

---

## サービス範囲一覧

| カテゴリ | 具体的内容 | 区分 | 備考・リスク |
|----------|------------|------|--------------|
| **身元保証・連絡** | 緊急連絡先としての登録 | ✅ | 24時間対応可否を契約で明示 |
| | 入退院・入退所時の連絡・同行 | ⚠️ | 範囲・回数の上限 |
| | 身元保証人としての記名・連帯保証 | ⚠️ | 法的責任の範囲を契約書で限定 |
| **費用・支払い** | 月次費用の支払い代行 | ⚠️ | 預託金と運転資金の分別管理 |
| | 入院・入所時の一時金・預り金の立替 | ⚠️ | 上限額・返金条件 |
| | 未払い・残債の保証 | ❌ | 連帯保証の範囲外とするか要法務確認 |
| **医療・介護** | 診療・治療方針の説明同席 | ✅ | **同意権限は持たない** |
| | 医療同意・手術同意の代行 | ❌ | 本人意思が基本。第三者に同意権限なし |
| | 介護サービス計画への意見表明 | 🔗 | ケアマネ・本人中心 |
| **意思決定支援** | 契約時の意思能力確認の支援 | 🔗 | 医師・弁護士・成年後見と連携 |
| | 認知症の方の新規契約 | ⚠️ | 判断能力・成年後見の有無で個別判断 |
| | 重要事項説明の同席・わかりやすい言葉での補足 | ✅ | 説明記録を残す |
| **入退院・入退所** | 手続き同行・書類作成支援 | ⚠️ | 行政書士領域は🔗 |
| | 退院・退所後の住居・施設手配 | ⚠️ | 成功保証はしない |
| **日常生活支援** | 安否確認・見守り訪問 | ⚠️ | 頻度・時間帯を契約化 |
| | 買い物・通院同行 | ⚠️ | 別プラン・都度課金も可 |
| **死後事務** | 身柄引取り連絡・葬儀手配の仲介 | ⚠️ | 事前契約・預託金必須 |
| | 遺品整理・家財処分 | ⚠️ | 業者🔗との役割分担 |
| | 相続・名義変更 | 🔗 | 司法書士・弁護士 |
| | 除票・年金・各種解約手続き | ⚠️ | 代行範囲を契約で限定 |
| **預託金・契約** | 預託金の受領・管理・精算 | ⚠️ | 分別管理・定期報告 |
| | 重要事項説明書の交付 | ✅ | 消費者庁ガイドライン準拠 |
| | 解約・返金 | ✅ | 規定を事前明示 |
| **その他** | 寄附・遺贈の受領・勧誘 | ❌ | 利益相反。原則禁止 |
| | 個人情報の第三者提供 | ⚠️ | 同意・目的限定 |

## 顧客別の提供イメージ（要ヒアリングで確定）

| 顧客 | 想定ニーズ | 課金イメージ |
|------|------------|--------------|
| 本人（利用者） | 連絡先・支払い代行・死後事務 | 月額＋預託金 |
| 施設 | 連絡・費用回収の負担軽減 | 紹介料／施設契約は要検討 |
| 病院 | 入院時連絡・説明同席 | B2B契約は要検討 |
| ケアマネ・紹介者 | 行き先・連絡先の確保 | 紹介料は法令・倫理確認 |

---

## 契約時に必ず説明する項目（チェック）

- [ ] サービス内容の一覧（本表に基づく）
- [ ] 費用（入会金・月額・都度・預託金）
- [ ] 支払方法・預託金の管理方法
- [ ] 判断能力低下時の対応方針
- [ ] 解約・返金条件
- [ ] 対応しないこと（医療同意代行など）
- [ ] 個人情報の取扱い
- [ ] 相談窓口・苦情対応
