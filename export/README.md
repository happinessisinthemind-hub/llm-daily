# 高齢者支援事業 — まとめ一式（Notion・Obsidian 共通）

**このフォルダの1ファイルだけ**で、Notion と Obsidian の両方に入れられます。

## ファイル

```
export/高齢者支援事業-まとめ一式.md
```

## Obsidian に入れる

1. ファイルを開いて **全文コピー**
2. Obsidian で新規ノート → **貼り付け**
3. または `.md` を iCloud Vault に保存

目次の `[[#パート1...]]` から各セクションへジャンプできます。

## Notion に入れる

1. Notion で新規ページ「高齢者支援事業」を作成
2. `···` → **Import** → **Markdown**
3. `高齢者支援事業-まとめ一式.md` を選択

## 含まれる内容

| パート | 内容 |
|--------|------|
| パート1 | 事業設計整理書（15項目・市場調査まとめ） |
| パート2 | 面談当日チートシート（30分用） |
| パート3 | 初月成果物5点 |

## 個別ノートが欲しい場合

- Obsidian（リンク付き個別版）: `obsidian/高齢者支援事業/`
- Notion（ページ分割用）: `notion/高齢者支援事業/`
